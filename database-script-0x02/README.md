# Database Seed Script (0x02)

This script populates the database with realistic sample data for testing.

## Data Included

- 3 users: guest, host, admin
- 2 properties
- 1 booking
- 1 payment
- 1 review
- 1 message

## Usage

Run after `schema.sql`:

```sql
SOURCE database-script-0x02/seed.sql;
```
